# Features
  - Vue + Vue Router
  - Vue tabs
  - Bootstrap system grid
  - Components UI
  - Dynamic imports layouts 
  - Nuxtjs loading  bar

## Usage

#### Development

```bash
# Build and watch
npm run dev

# Run tests with jest
npm test:watch
```

#### Production

```bash
npm run build
```
